# 🎮 游戏大厅 - 经典小游戏合集

5款经典HTML5小游戏合集，支持PWA安装，可离线游玩，完美适配移动端。

## 🎯 包含游戏

### 🐍 贪吃蛇
经典贪吃蛇游戏，控制蛇吃食物，避免撞墙和咬到自己。

**特性**：
- 20×20网格，格子清晰可见
- 流畅的操作体验
- 触摸滑动控制
- 最高分记录
- 暂停功能

### 🧱 俄罗斯方块
经典益智消除游戏，旋转和移动方块进行消除。

**特性**：
- 10×20网格，移动端完美适配
- 7种经典方块
- 等级系统
- 连消加分
- 触摸滑动控制

### 💎 消消乐
三消益智游戏，交换相邻宝石进行消除。

**特性**：
- 8×8网格，触摸点击精准
- 6种颜色宝石
- 30步限制
- 连锁反应
- 自动下落填充

### 💣 扫雷
经典逻辑推理游戏，标记地雷安全通关。

**特性**：
- 16×16网格，格子大小适中
- 40个地雷
- 点击揭开，长按标记
- 自动展开空白区域
- 触摸优化

### ♟️ 五子棋
人机对战五子棋，五子连珠即可获胜。

**特性**：
- 15×15标准棋盘
- 简单AI对手
- 触摸落子
- 黑白棋立体效果
- 经典木纹棋盘

## 📱 安装方法

### Android
1. 在浏览器打开游戏网址
2. 点击浏览器菜单
3. 选择"添加到主屏幕"
4. 点击"安装"

### iPhone
1. 在Safari打开游戏网址
2. 点击底部分享按钮
3. 选择"添加到主屏幕"
4. 点击"添加"

## 🎮 操作说明

### 贪吃蛇
- 方向键/WASD：控制方向
- 空格键：暂停/继续
- 触摸滑动：控制方向
- 按钮控制：方向按钮

### 俄罗斯方块
- 左右方向键：移动
- 上方向键/W：旋转
- 空格键：快速下落
- P键：暂停
- 触摸滑动：左右移动，上滑旋转，下滑快速下落

### 消消乐
- 点击选择宝石
- 再次点击相邻宝石交换
- 三个或更多相同颜色连线即可消除

### 扫雷
- 点击：揭开格子
- 长按（500ms）：标记地雷
- 数字表示周围地雷数量

### 五子棋
- 点击棋盘交叉点落子
- 黑棋先行（你）
- 五子连珠即可获胜
